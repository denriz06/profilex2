## 📋 Deskripsi Project

Website ini adalah portfolio demo milik **Raden Fariz Attila Ahyad**, alumni SMK Wikrama Bogor jurusan Rekayasa Perangkat Lunak. Dibuat dengan desain terinspirasi dari game **Valorant** — animasi partikel, warna merah khas, dan UI dengan sudut tajam (clip-path).

## 🛠️ Teknologi

- **HTML5** — Struktur halaman
- **CSS3** — Styling, animasi, clip-path, responsive
- **JavaScript (Vanilla)** — Logic SPA, game engine, interaksi

> Tidak menggunakan framework atau library eksternal (kecuali Google Fonts).

## 🚀 Cara Pakai

### Buka Langsung
Cukup buka file `index.html` di browser — tidak perlu server.

### Upload ke GitHub Pages
1. Buat repository baru di GitHub
2. Upload `index.html` dan `README.md`
3. Masuk ke **Settings → Pages → Source: main branch**
4. Website langsung live di `https://username.github.io/nama-repo`
