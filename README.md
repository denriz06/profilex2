---

## 📁 Struktur File

```
profilex2/
├── index.html   ← Struktur HTML semua halaman
├── style.css    ← Semua styling & animasi
├── data.js      ← Data quotes, skills, ranking, wordlist game
├── theme.js     ← Sistem 4 tema warna
├── game.js      ← Engine Type Game (requestAnimationFrame)
├── main.js      ← Navigasi, particles, render halaman
└── README.md    ← Dokumentasi ini
```

> Urutan load script di index.html: `data.js` → `theme.js` → `game.js` → `main.js`

---

## 🚀 Deploy ke GitHub Pages

1. Upload semua file ke **root** repository (bukan dalam subfolder)
2. Settings → Pages → Source: **main branch / root**
3. Tunggu 1–2 menit → live di `https://username.github.io/nama-repo`
